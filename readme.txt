The b2zone-Shoplet Plugin allows you to embed a Shoplet in WordPress and sell products directly from your WordPress site.

== Description ==

The b2zone-Shoplet Plugin allows you to embed a Shoplet in WordPress and sell products directly from your WordPress site. After the plugin installation it is possible to embed a Shoplet using [shoplet][/shoplet]-tags.

== Installation ==

1. Upload `wp-shoplet-plugin.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place `[shoplet]<your b2zone-Shoplet ID>[/shoplet]` in your posts or pages.

== Frequently Asked Questions ==

= What is a Shoplet? =

A Shoplet represents your product on the web and it enables you to easily offer your product online, no matter if it is a physical or electronic (file) product.

= There are two types of delivery: physical and electronic delivery. What is the difference? =

If you choose physical delivery, you sell a physical object which you have to ship to the buyer. You choose electronic delivery if you want to sell an electronic file. For example: an e-book or a music album.
